# O2Kml
KML Serializer using Generic XMLSerializer

## Basic Usage
A KML file can be deserialized by assigning the result of O2Kml.Kml.LoadFromFile(...) or TryLoadFromFile(...)
To serialize, call Kml.SaveToFile(...)
